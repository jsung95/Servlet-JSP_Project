# Servlet/JSP Web Simple Board Project

-------

## ■ 개요
- 본 프로젝트는 html5up.net HTML5 공개 템플릿을 기반으로 제작.
- **Servlet/JSP**로 작성된 프로젝트에 MySQL JDBC를 이용하여 데이터베이스에 연결하고, **데이터 C.R.U.D** 구현을 목표로한다.
- **MVC 패턴**을 기반으로 프로젝트 구성.
